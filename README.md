# 裂星纪元 (Rift Star Era)

基于 HTML Canvas 的即时战略 (RTS) 网页游戏，支持单人对 AI 和局域网联机对战。

## 快速开始

用浏览器直接打开 `index_c5e9607a.html` 即可游玩单人模式。

联机对战需要启动 WebSocket 服务器：

```bash
node server.js
```

默认端口 3000，可通过环境变量 `PORT` 修改。

## 阵营

游戏有三个阵营，各有独特的兵种和科技路线：

| 阵营 | 特色 | 优势 |
|------|------|------|
| 星火族 | 爆发 · 溅射 | 高攻击力，范围伤害强 |
| 幽影族 | 隐形 · 控制 | 潜行突袭，虚空诅咒减速 |
| 钢铁联军 | 重甲 · 阵地战 | 高护甲高血量，工人负载最大 |

## 操作方式

### 鼠标/触屏

| 操作 | 说明 |
|------|------|
| 左键单击 | 选中单位或建筑 |
| 左键拖拽 | 框选多个单位 |
| 右键点击空地 | 移动选中单位 |
| 右键点击敌人 | 攻击目标 |
| 右键点击资源 | 工人前往采集 |
| 右键点击建造中建筑 | 工人协助建造 |
| 鼠标滚轮 | 缩放视角 |
| 鼠标移至屏幕边缘 | 卷动视角 |
| 点击小地图 | 快速跳转视角 |

### 键盘快捷键

| 按键 | 功能 |
|------|------|
| `Esc` / `P` | 暂停游戏（单人模式） |
| `Esc` | 取消建造模式 / 取消选中 |
| `S` | 停止选中单位当前动作 |
| `空格` | 镜头回到主基地 |
| `F1` | 显示/隐藏帮助面板 |
| `A` | 攻击移动提示 |

## 游戏流程

### 1. 选择阵营和难度

启动后选择阵营（星火族 / 幽影族 / 钢铁联军），选择难度（简单 / 普通 / 困难 / 疯狂），点击"单人游戏"。

### 2. 发展经济

- 主基地生产**工人**，派工人采集**木材**、**食物**和**金币**
- 建造**补给站**提高人口上限
- 建造**采集场**作为离基地较远的资源投递点

### 3. 建造军事建筑

- **兵营** → 基础兵种（战士、射手）
- **工坊** → T2 重型单位（坦克、骑兵、攻城车）— 需要 T2 时代
- **法师塔** → T2 法系单位（法师、术士）— 需要 T2 时代
- **机场** → T2 飞行单位（蝠龙、旋翼机）— 需要 T2 时代
- **终极建筑** → T3 终极单位（凤凰、炎魔、泰坦）— 需要 T3 时代 + 工坊

### 4. 研发科技

建造**研究室**后可研发科技：
- 通用科技：近战/远程攻击提升、护甲加固、负载提升、金币采集效率
- 阵营专属：解锁高级兵种、特殊能力

### 5. 进攻

集结部队后右键点击敌方建筑或单位发起进攻。摧毁敌方**主基地**即可获胜。

## 资源类型

| 资源 | 来源 | 用途 |
|------|------|------|
| 木材 🌲 | 砍伐森林 | 建筑、部分兵种 |
| 食物 🌾 | 食物田（2×2） | 工人、大部分兵种 |
| 金币 💰 | 金矿（稀有） | 高级兵种、科技 |

## 时代升级

| 时代 | 解锁时间 (AI) | 解锁内容 |
|------|-------------|---------|
| T1 | 游戏开始 | 基础兵种、兵营、补给站 |
| T2 | 3 分钟 | 工坊、法师塔、机场及其兵种 |
| T3 | 6 分钟 | 终极建筑及终极兵种 |

## 联机对战

1. 启动服务器 `node server.js`
2. 在游戏主界面点击"联机对战"
3. 一方点击"创建房间"获得 4 位房间码
4. 另一方输入房间码点击"加入房间"
5. 双方准备后自动开始

## 项目结构

```
index_c5e9607a.html   # 游戏主入口（完整单文件版）
server.js             # WebSocket 联机服务器
js/                   # 模块化源码
  game-data.js        # 阵营、建筑、科技数据
  entity-factory.js   # 实体创建与索引
  ai.js               # AI 决策系统
  ai-logic.js         # AI 辅助逻辑
  combat.js           # 战斗系统（伤害、状态机、迷雾）
  movement.js         # 移动、寻路队列、单位分离
  world.js            # 地图生成、A* 寻路
  input.js            # 输入处理（鼠标、触屏、框选）
  render.js           # 渲染系统
  render-logic.js     # 渲染辅助逻辑
  ui.js               # UI 面板更新
  network.js          # 网络通信
  sprite-loader.js    # 精灵图加载
tests/                # 单元测试（node 运行）
assets/sprites/       # 精灵图资源
docs/plans/           # 开发计划文档
```
